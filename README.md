 Projet Vente - Site e-commerce
Ce projet consiste en la création d’un site de vente en ligne réalisé en HTML et CSS dans le cadre d’un projet académique.

 Objectif
Concevoir un site web complet pour une association avec un design moderne, une bonne ergonomie et une accessibilité optimisée.

 Fonctionnalités
- Page d’accueil avec produits et catégories
- Pages catégories (plusieurs)
- Pages descriptives des produits
- Page panier / support avec formulaire de contact
- Page erreur 404
- Navigation complète entre les pages
- Design responsive (mobile, tablette, desktop)

  Technologies utilisées
- HTML5
- CSS3

  Responsive Design
Le site est compatible avec :
- Ordinateur (desktop)
- Tablette
- Mobile

  Accessibilité
Respect des bonnes pratiques d’accessibilité (RGAA / WCAG) :
- Structure HTML claire
- Bon contraste des couleurs
- Navigation simple
- 
 Structure du projet
- index.html
- pages catégories
- pages produits
- page contact / panier
- page 404
- dossier CSS
- dossier images
- 
 Lancer le projet
Ouvrir le fichier `index.html` dans un navigateur.
